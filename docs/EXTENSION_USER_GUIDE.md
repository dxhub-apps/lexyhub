## Getting Started

### Installation

1. Install the LexyHub Extension from the Chrome Web Store
2. Click the extension icon in your browser toolbar
3. Sign in with your LexyHub account
4. Start browsing supported marketplaces!

### First-Time Setup

After installation, the extension will:
- Open a welcome page with setup instructions
- Initialize default settings for all supported domains
- Enable keyword highlighting and tooltips by default

## Using the Popup Dashboard

Click the extension icon in your toolbar to open the dashboard with 4 tabs:

### 1. Discover Tab
**See what's trending right now**

- View top trending keywords in your selected market
- Click any keyword to copy it
- Use the "Create Brief" button for deeper analysis
- Filter by marketplace (Etsy, Amazon, Shopify, etc.)

**Pro Tip:** Check the Discover tab daily to spot emerging trends before they become saturated!

### 2. Session Tab
**Track your current research session**

View real-time stats:
- **Queries** - Number of searches you've performed
- **Clicks** - Listings you've clicked on
- **Discovered** - New keywords you've found
- **Duration** - How long you've been researching

**Actions:**
- View full session details
- End current session (auto-saves to your account)

**Pro Tip:** Sessions auto-save after 30 minutes of inactivity, so you never lose your research progress.

### 3. Briefs Tab
**Access your saved keyword briefs**

- View all generated briefs
- See creation date and keyword count
- Click to open full brief with:
  - Opportunity clusters
  - Executive summary
  - AI insights
  - Recommended actions

**Pro Tip:** Create briefs for related keywords together (e.g., "vintage jewelry", "handmade jewelry", "boho jewelry") to find cross-selling opportunities.

### 4. Settings Tab
**Quick access to key settings**

Toggle features on/off:
- Keyword highlighting
- Metric tooltips
- Auto-capture keywords
- Community data sharing

**Pro Tip:** If highlighting feels overwhelming, try toggling it off and using Ctrl+Shift+K to open quick actions instead.

## Context Menu Features

Right-click on any selected text to access LexyHub features:

### Add to Watchlist
1. Select any text on a webpage
2. Right-click and choose "Add '[text]' to LexyHub Watchlist"
3. The keyword is instantly saved with source URL
4. Receive a notification confirming the save

### Create Brief
1. Select any text on a webpage
2. Right-click and choose "Create LexyHub Brief for '[text]'"
3. AI begins generating a comprehensive brief
4. Access it from the Briefs tab when ready

**Pro Tip:** You can select multiple keywords separated by commas to create multi-keyword briefs!

## Settings & Configuration

Access full settings by right-clicking the extension icon and selecting "Options" or clicking "Settings" in the popup.

### Domain Settings

Enable/disable the extension for each marketplace:

- **Etsy** - Etsy marketplace keyword detection
- **Amazon** - All Amazon domains (.com, .ca, .co.uk, etc.)
- **Shopify** - Shopify stores (auto-detects)
- **Google** - Google Search results
- **Bing** - Bing Search results
- **Pinterest** - Pinterest boards and pins
- **Reddit** - Reddit posts and communities

**Pro Tip:** Disable domains you don't use to improve performance.

### Feature Settings

Fine-tune how the extension works:

- **Keyword Highlighting** - Visual underlining of detected keywords
- **Metric Tooltips** - Show/hide hover tooltips
- **Auto-Capture** - Automatically save discovered keywords
- **Smooth Animations** - Enable/disable UI transitions

### Display Preferences

- **Highlight Color** - Choose your preferred highlight color
- **Tooltip Delay** - Adjust how quickly tooltips appear (in milliseconds)

### Privacy Settings

- **Community Signals** - Opt-in to anonymous trend data sharing
  - When enabled, keywords you discover contribute to trending data
  - No personal information is shared
  - Helps the community discover emerging opportunities
  - Can be disabled at any time

### Account Management

View your account information:
- Current plan (Free/Pro/Enterprise)
- Keywords tracked
- Briefs created
- Session count

**Actions:**
- **Export Data** - Download all your extension data as JSON
- **Delete All Data** - Permanently remove all extension data and reset settings

**Warning:** Deleting data is permanent and cannot be undone!

## Troubleshooting

### Extension Not Working

**Check these first:**
1. Ensure you're logged in (click extension icon)
2. Verify the domain is enabled in Settings
3. Refresh the page after enabling a domain
4. Check that you're on a supported marketplace

### Tooltips Not Appearing

1. Go to Settings ‚Üí Feature Settings
2. Ensure "Metric Tooltips" is enabled
3. Check "Tooltip Delay" isn't set too high (try 500ms)
4. Refresh the page

### Keywords Not Highlighting

1. Go to Settings ‚Üí Feature Settings
2. Enable "Keyword Highlighting"
3. Refresh the page
4. Try hovering over product titles or search terms

### Session Not Tracking

1. Open the popup ‚Üí Session tab
2. Click "End Session" then navigate to a marketplace
3. Sessions auto-start when you perform a search
4. Check that the domain is enabled in Settings

### Briefs Not Generating

1. Ensure you're on a Pro or Enterprise plan
2. Check your brief quota hasn't been exceeded
3. Try with 1-3 keywords first (don't exceed 10)
4. Wait 10-15 seconds for AI generation

### Performance Issues

**If the extension slows down your browser:**
1. Disable domains you don't use
2. Turn off "Smooth Animations"
3. Disable "Auto-Capture"
4. Clear extension data: Settings ‚Üí Delete All Data

### Login Issues

1. Click "Login" in the popup
2. Complete authentication on lexyhub.com
3. Return to the extension
4. If still logged out, try removing and reinstalling the extension

## FAQ

### General Questions

**Q: Is the extension free?**
A: Yes! Core features are free forever. Pro features like AI briefs require a paid plan.

**Q: Does it work on mobile browsers?**
A: Currently Chrome desktop only. Mobile support coming soon.

**Q: Can I use it on Firefox or Safari?**
A: Chrome/Edge only right now. More browsers coming based on demand.

**Q: Does it slow down my browser?**
A: No. The extension is optimized for minimal performance impact. If you experience issues, see [Performance Issues](#performance-issues).

### Features

**Q: How accurate are the keyword metrics?**
A: Metrics are refreshed daily from multiple data sources. Freshness indicator shows data age.

**Q: What's the AI Score based on?**
A: It combines demand, competition, engagement, trend, and seasonality into a 0-100 opportunity score.

**Q: Can I export my watchlist?**
A: Yes! Settings ‚Üí Account ‚Üí Export Data downloads everything as JSON.

**Q: Do sessions track across different tabs?**
A: Yes! Sessions are unified across all marketplace tabs until you close them all or 30 minutes of inactivity passes.

**Q: How many briefs can I generate?**
A: Free: 5/month, Pro: 50/month, Enterprise: Unlimited

### Privacy

**Q: Does the extension track my browsing?**
A: No. We only capture keywords you interact with on supported marketplaces.

**Q: What are "community signals"?**
A: Anonymous, aggregated data showing how many users discovered a keyword recently. Opt-in only.

**Q: Can I delete all my data?**
A: Yes. Settings ‚Üí Delete All Data removes everything permanently.

**Q: Do you share data with marketplaces?**
A: Never. Your research stays private between you and LexyHub.

### Technical

**Q: Why do I need to grant so many permissions?**
A:
- `storage` - Save your settings locally
- `activeTab` - Detect keywords on current page
- `scripting` - Inject highlighting overlays
- `contextMenus` - Right-click menu actions
- `notifications` - Confirm saves
- `alarms` - Periodic config updates

**Q: Does it work offline?**
A: Highlighting works offline, but metrics require an internet connection.

**Q: Can I customize the highlight color?**
A: Yes! Settings ‚Üí Display Preferences ‚Üí Highlight Color

**Q: What's the rate limit?**
A: Free: 100 requests/minute, Pro: 200 requests/minute. You'll rarely hit this during normal use.

## Tips & Best Practices

### Research Workflow

1. **Start with Discover** - Check trending keywords in your niche
2. **Browse naturally** - Shop as your customer would
3. **Let tooltips guide you** - High AI scores = opportunities
4. **Save promising keywords** - Build your watchlist
5. **Review your session** - Analyze what you discovered
6. **Create briefs** - Deep-dive into top opportunities
7. **Export data** - Back up your research regularly

### Finding Hidden Gems

- Check "People Also Ask" on Google for long-tail questions
- Look at Reddit post titles in niche communities
- Browse Pinterest boards for visual search terms
- Review Amazon "Customers Also Bought" suggestions
- Analyze competitor product titles on Etsy

### Optimizing Performance

- Enable only the domains you actively research
- Use keyboard shortcuts instead of clicking
- Create briefs in batches (3-5 at a time)
- Export and clear old sessions monthly
- Keep your watchlist under 500 terms for best speed
